#### فئة 2 — اختبارات قديمة (19 فشل) — ⚠️ تحتاج تحديث

**`language_detector_test.dart` (14 فشل)**
الاختبارات تتوقع lowercase مثل `'python'` لكن الكود يُرجع `'Python'`.
الكود صحيح — الاختبارات لم تُحدَّث.

| الاختبار | المتوقع | الفعلي |
|----------|---------|--------|
| detects Python | `'python'` | `'Python'` |
| detects JavaScript | `'javascript'` | `'Lua'` (كود الاختبار خاطئ) |
| detects Java | `'java'` | `null` |
| detects Dart | `'dart'` | `'Dart'` |
| detects C++ | `'cpp'` | `'C++'` |
| + 9 اختبارات مشابهة | | |

**`editor_state_manager_test.dart` (5 فشل)**
الاختبارات تتوقع `hasChanges()` يُرجع `true` مباشرة بعد تغيير المحتوى،
لكن المنطق الحالي يتطلب `markDirty()` أولاً.
الكود صحيح — الاختبارات لم تُحدَّث.

| الاختبار | المشكلة |
|----------|----------|
| `hasChanges detects content changes` | لا يستدعي `markDirty()` |
| `updateSnapshot updates snapshot to current state` | لا يستدعي `markDirty()` |
| `updateSnapshot updates all snapshot fields` | لا يستدعي `markDirty()` |
| `Edge Cases handles empty content` | لا يستدعي `markDirty()` |
| `Complex Scenarios handles multiple changes` | لا يستدعي `markDirty()` |
