# سياق مشروع موقع Sinan Note

## المشروع الأصلي (Flutter App)
- **الاسم**: Sinan Note (سنان نوت)
- **الإصدار**: 2.3.0
- **المطور**: Apex Flow Group
- **المنصات**: Android (Google Play) + Linux + Windows
- **المستودع**: مجلد `apex_note` في `/home/dream/GitHub/Sinan_Note_G`

### مميزات التطبيق
- خزنة مشفرة AES-256 + مصادقة بيومترية
- محرر كود بـ 25+ لغة برمجية مع syntax highlighting
- أنواع ملاحظات: نص بسيط، كود، قوائم مهام، تذكيرات، rich text
- ثنائي اللغة عربي/إنجليزي مع RTL/LTR تلقائي
- Material You + وضع ليلي/نهاري
- مزامنة Google Drive + نسخ احتياطي JSON
- تخطيط Master-Details لسطح المكتب
- تذكيرات محلية مع recurrence rules
- تاريخ إصدارات الملاحظات
- Home Widget للشاشة الرئيسية
- بحث ذكي مع دعم عربي (تطبيع الحروف)

### Stack التطبيق
- Flutter 3.0+ / Dart 3.0+
- Isar (قاعدة بيانات محلية)
- Provider (إدارة الحالة)
- Clean Architecture

## مشروع الموقع (SinanAi.net)

### المسار
`/home/dream/GitHub/SinanAi.net`

### التقنية المختارة
- **Framework**: Next.js
- **Styling**: CSS (لا Tailwind — HTML/CSS نظيف)
- **Hosting**: Amazon (سيرفر متوفر)

### محتوى الموقع المطلوب
1. **تعريف بالتطبيق** — المميزات، لقطات الشاشة، وصف كامل
2. **تعريف بالمطور** — Apex Flow Group
3. **روابط** — Google Play، المستودع، إلخ
4. **شرح Fork المشروع** — كيف تعمل fork من GitHub
5. **شرح التحميل المباشر** — تحميل APK/نسخة من الموقع مباشرة
6. **أدوات بناء نسخة** — شرح كيف تبني من المستودع (Flutter build guide)

### المتطلبات التقنية
- **i18n**: متعدد اللغات — يبدأ بعربي + إنجليزي، قابل للتوسع
- **Theming**: سيمات متعددة قابلة للتخصيص
- **بنية قابلة للتوسع**: مفهومة وسهلة الإضافة
- **جودة عالية**: ليس موقعاً بدائياً — متكامل واحترافي
- **RTL/LTR**: دعم كامل للعربية

### القرارات المتخذة
- نبدأ بوثيقة المتطلبات أولاً (Requirements-First)
- السيرفر Amazon — البنية تراعي هذا (SSR/SSG/deployment)
- لا نقبل بأقل من موقع متكامل شامل
