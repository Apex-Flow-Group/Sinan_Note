## Installation

### For Users:
```bash
# Download APK from releases
# Or build from source:
flutter build apk --release
```

### For Developers:
```bash
git clone https://github.com/apexflow/sinan-note.git
cd sinan-note
flutter pub get
flutter run
```

## Features

### 🔐 Security
- AES-256 encryption for locked notes
- Biometric authentication
- Temporary session management (5 minutes)
- Memory wipe on exit

### 📝 Note Types
- Simple notes
- Professional code editor (20+ languages)
- Checklists with progress tracking
- Reminders with recurrence

### 🎨 User Experience
- Material Design 3
- Dynamic colors (Material You)
- RTL/LTR support
- Dark/Light themes
- Flexible views (Grid/List)

### 🔄 Backup & Sync
- Local JSON export/import
- Database backup/restore
- Google Drive (coming soon)

**See [README_EN.md](docs/README_EN.md) for complete feature list.**
