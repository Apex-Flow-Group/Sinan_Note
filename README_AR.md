# 📝 سنان نوت | Sinan Note

<div dir="rtl">

## نظرة عامة

**سنان نوت** هو تطبيق تدوين ملاحظات ذكي مصمم بفلسفة **السرعة، الأمان، والبساطة**. تم بناؤه باستخدام Flutter لتوفير تجربة سلسة عبر منصات Android وLinux وWindows.

> **"سنان"** تعني الحدة والدقة - تماماً كما يجب أن تكون أداة التدوين المثالية.

### 🎯 الرؤية

- **⚡ السرعة**: واجهة سريعة الاستجابة مع حفظ تلقائي فوري
- **🔒 الأمان**: خزنة ذكية بتشفير AES-256 مع جلسات مؤقتة
- **🎨 البساطة**: تصميم نظيف يدعم RTL/LTR مع Material Design 3

---

## ✨ المميزات الرئيسية

### 🔐 الخزنة الذكية (Smart Vault)
- **تشفير AES-256**: حماية عسكرية لملاحظاتك الحساسة
- **جلسة مؤقتة**: فتح الخزنة لمدة 5 دقائق فقط، ثم قفل تلقائي
- **مسح الذاكرة**: تنظيف RAM فوري عند الخروج من التطبيق أو الانتقال للخلفية
- **مصادقة بيومترية**: دعم بصمة الإصبع/الوجه (حسب الجهاز)

### 📝 أنواع الملاحظات
1. **ملاحظات بسيطة**: للتدوين السريع اليومي
2. **ملاحظات احترافية**: محرر كود متقدم مع Syntax Highlighting لـ 20+ لغة برمجة
3. **قوائم المهام**: Checklists تفاعلية مع تتبع التقدم
4. **التذكيرات**: جدولة إشعارات ذكية مع تكرار (يومي/أسبوعي/شهري)

### 🚀 الأداء والتجربة
- **حفظ تلقائي**: لا حاجة للضغط على "حفظ" - كل شيء يُحفظ فوراً
- **بحث ذكي**: فلترة حسب النوع، الحالة، أو النص
- **عرض مرن**: شبكة (Grid) أو قائمة (List) موسعة/مضغوطة
- **Slivers Performance**: قوائم سلسة حتى مع آلاف الملاحظات
- **دعم RTL/LTR**: تبديل سلس بين العربية والإنجليزية

### 🎨 التخصيص
- **ألوان ديناميكية**: Material You (Android 12+)
- **الوضع الليلي**: تبديل تلقائي أو يدوي
- **حجم النص**: قابل للتعديل (0.8x - 1.5x)
- **تعتيم الخلفية**: حماية الخصوصية عند التبديل بين التطبيقات

### 🔄 النسخ الاحتياطي والمشاركة
- **تصدير/استيراد**: JSON backup محلي
- **Google Drive**: مزامنة سحابية (قريباً)
- **نقل P2P**: مشاركة الملاحظات بين الأجهزة عبر WiFi Direct
- **مشاركة سريعة**: تصدير إلى أي تطبيق

### 📱 ميزات إضافية
- **الأرشيف**: إخفاء الملاحظات القديمة دون حذفها
- **سلة المهملات**: استعادة الملاحظات المحذوفة خلال 30 يوم
- **التثبيت**: تثبيت الملاحظات المهمة في الأعلى
- **الويدجت**: عرض الملاحظات على الشاشة الرئيسية (Android)
- **استقبال الملفات**: فتح ملفات نصية مباشرة من تطبيقات أخرى

## 🛠️ التثبيت والبناء

### المتطلبات
- Flutter SDK 3.0.0 أو أحدث
- Dart SDK 3.0.0 أو أحدث
- Android Studio / VS Code (للتطوير)

### بناء نسخة Android

#### 1. تثبيت الاعتماديات
```bash
flutter pub get
```

#### 2. بناء APK
```bash
# نسخة Release (موقعة)
flutter build apk --release

# نسخة Debug (للاختبار)
flutter build apk --debug
```

#### 3. بناء App Bundle (للنشر على Google Play)
```bash
flutter build appbundle --release
```

**موقع الملف الناتج:**
- APK: `build/app/outputs/flutter-apk/app-release.apk`
- Bundle: `build/app/outputs/bundle/release/app-release.aab`

### بناء نسخة Linux
```bash
flutter build linux --release
```

### بناء نسخة Windows
```bash
flutter build windows --release
```

### التشغيل السريع (للتطوير)
```bash
# Android
flutter run

# Linux
flutter run -d linux

# Windows
flutter run -d windows
```

## 🏗️ الهيكلية التقنية

المشروع مبني على **Clean Architecture** مع فصل واضح للمسؤوليات:

```
lib/
├── models/          # نماذج البيانات (Note, NoteMode, Exceptions)
├── services/        # منطق الأعمال (Providers, Database, Encryption)
├── screens/         # واجهات المستخدم (Home, Editor, Settings)
├── widgets/         # مكونات قابلة لإعادة الاستخدام
├── utils/           # أدوات مساعدة
└── l10n/            # الترجمات (AR/EN)
```

**للمزيد من التفاصيل التقنية، راجع:** [ARCHITECTURE.md](ARCHITECTURE.md)

## 📚 دليل الاستخدام

للحصول على شرح مفصل لجميع الميزات، راجع: [USER_MANUAL.md](USER_MANUAL.md)

### البدء السريع

1. **إنشاء ملاحظة جديدة**
   - اضغط على زر `+` في الأسفل
   - اختر نوع الملاحظة (بسيطة/احترافية/قائمة مهام)

2. **استخدام الخزنة**
   - افتح القائمة الجانبية → "الخزنة"
   - أدخل بصمتك/رمز PIN
   - الجلسة تنتهي تلقائياً بعد 5 دقائق

3. **البحث والفلترة**
   - اضغط على شريط البحث
   - استخدم أيقونة الفلتر لتصفية حسب النوع

4. **النسخ الاحتياطي**
   - الإعدادات → النسخ الاحتياطي
   - اختر "تصدير" لحفظ نسخة محلية

## 🔧 التكوين

### ملف `pubspec.yaml`
يحتوي على جميع الاعتماديات الأساسية:
- `provider`: إدارة الحالة
- `sqflite`: قاعدة البيانات المحلية
- `encrypt`: تشفير AES-256
- `flutter_local_notifications`: التذكيرات
- `local_auth`: المصادقة البيومترية
- `flutter_code_editor`: محرر الكود

### ملف `android/key.properties`
لتوقيع التطبيق (غير مضمن في Git):
```properties
storePassword=<your_password>
keyPassword=<your_password>
keyAlias=sinan-release-key
storeFile=sinan-release-key.jks
```
