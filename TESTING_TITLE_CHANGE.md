## 🧪 خطوات الاختبار

### اختبار 1: المحرر البسيط (Simple Note)
```
1. افتح ملاحظة بسيطة جديدة
2. اكتب بعض النص
3. انقر على العنوان في الأعلى
4. غيّر العنوان إلى "اختبار 1"
5. اضغط حفظ
6. تحقق: العنوان تغير في الـ Header
7. اخرج من الملاحظة
8. افتحها مرة أخرى
9. تحقق: العنوان محفوظ بشكل صحيح
```

### اختبار 2: محرر الكود (Code Editor)
```
1. افتح ملاحظة كود جديدة
2. اكتب كود Python بسيط
3. انقر على العنوان
4. غيّر العنوان إلى "Python Script"
5. اضغط حفظ
6. تحقق: العنوان تغير
7. غيّر الكود
8. اخرج (يجب أن يحفظ تلقائياً)
9. افتح الملاحظة مرة أخرى
10. تحقق: العنوان والكود محفوظان
```

### اختبار 3: قائمة المهام (Checklist)
```
1. افتح قائمة مهام جديدة
2. أضف بعض المهام
3. انقر على العنوان
4. غيّر العنوان إلى "مهام اليوم"
5. اضغط حفظ
6. تحقق: العنوان تغير
7. أضف مهمة جديدة
8. اخرج
9. افتح القائمة مرة أخرى
10. تحقق: العنوان والمهام محفوظة
```

### اختبار 4: ملاحظة مع تذكير (Reminder)
```
1. افتح ملاحظة تذكير جديدة
2. اكتب نص التذكير
3. أضف تاريخ ووقت
4. انقر على العنوان
5. غيّر العنوان إلى "موعد مهم"
6. اضغط حفظ
7. تحقق: العنوان تغير
8. اخرج وافتح مرة أخرى
9. تحقق: العنوان والتذكير محفوظان
```

### اختبار 5: نظام الحفظ الذكي
```
1. افتح ملاحظة موجودة
2. انقر على العنوان
3. غيّر العنوان فقط (بدون تغيير المحتوى)
4. اضغط حفظ
5. تحقق في الـ Debug Console:
   - يجب أن يحفظ (لأن العنوان تغير)
   - لا يجب أن يظهر "Save skipped"
6. افتح الملاحظة مرة أخرى بدون تغيير
7. اخرج مباشرة
8. تحقق في الـ Debug Console:
   - يجب أن يظهر "Save skipped: No real changes detected"
```

### اختبار 6: العنوان الفارغ
```
1. افتح ملاحظة جديدة
2. انقر على العنوان
3. امسح العنوان بالكامل
4. اضغط حفظ
5. تحقق: يجب أن يعود للعنوان التلقائي (أول سطر من المحتوى)
```

### اختبار 7: الملاحظات المقفلة
```
1. افتح ملاحظة مقفلة
2. أدخل البصمة/كلمة المرور
3. انقر على العنوان
4. غيّر العنوان
5. اضغط حفظ
6. اخرج وأعد الفتح
7. تحقق: العنوان محفوظ ومشفر بشكل صحيح
```

## 🐛 مشاكل محتملة وحلولها

### المشكلة: العنوان لا يُحفظ
**الحل:** تحقق من أن `_isDirty = true` يتم تعيينه عند تغيير العنوان

### المشكلة: نظام الحفظ الذكي لا يكتشف التغيير
**الحل:** تحقق من أن `hasTitleChanged` موجود في شرط الفحص

### المشكلة: العنوان يختفي بعد إعادة الفتح
**الحل:** تحقق من أن `_originalTitle` يتم تحديثه في `isManualSave`

## 🎯 الخلاصة

الميزة مطبقة بشكل صحيح في الكود ويجب أن تعمل في جميع أنواع المحررات. 

للاختبار الفعلي:
1. قم بتشغيل التطبيق: `flutter run`
2. اتبع خطوات الاختبار أعلاه
3. راقب الـ Debug Console للتحقق من رسائل الحفظ
4. تأكد من أن العنوان يُحفظ ويُسترجع بشكل صحيح

**ملاحظة:** إذا واجهت أي مشكلة، تحقق من الـ Debug Console لرؤية رسائل `debugPrint` التي تشير إلى سلوك نظام الحفظ الذكي.
