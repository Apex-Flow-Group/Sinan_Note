# قائمة اختبار الميزات الجديدة | New Features Test Checklist

## ✅ اختبار "عمل نسخة" | Test "Make a Copy"

### الخطوات:
1. [ ] افتح أي ملاحظة موجودة
2. [ ] اضغط على قائمة الخيارات (⋮) في الأعلى
3. [ ] اختر "عمل نسخة" / "Make a Copy"
4. [ ] تحقق من ظهور رسالة نجاح
5. [ ] ارجع للقائمة الرئيسية
6. [ ] تأكد من وجود نسخة جديدة بعنوان "العنوان - Copy"

### النتيجة المتوقعة:
- ✅ نسخة جديدة تم إنشاؤها
- ✅ المحتوى مطابق للأصل
- ✅ التاريخ جديد
- ✅ لا يوجد تذكير في النسخة
- ✅ النسخة غير مثبتة

## ✅ اختبار "حفظ باسم" | Test "Save As"

### الخطوات:
1. [ ] افتح أي ملاحظة موجودة
2. [ ] اضغط على قائمة الخيارات (⋮)
3. [ ] اختر "حفظ باسم" / "Save As"
4. [ ] أدخل عنوان جديد (مثل: "نسخة تجريبية")
5. [ ] اضغط "حفظ" / "Save"
6. [ ] تحقق من رسالة النجاح
7. [ ] ارجع للقائمة الرئيسية
8. [ ] تأكد من وجود النسخة بالعنوان الجديد

### النتيجة المتوقعة:
- ✅ نسخة جديدة بالعنوان المخصص
- ✅ المحتوى مطابق للأصل
- ✅ الملاحظة الأصلية لم تتغير

## ✅ اختبار "فتح ملفات برمجية" | Test "Open Programming Files"

### الخطوات - Python:
1. [ ] أنشئ ملف test.py على الجهاز بمحتوى:
```python
def hello():
    print("Hello from Sinan Note!")
```
2. [ ] افتح مدير الملفات
3. [ ] اضغط على test.py
4. [ ] اختر "فتح بواسطة" / "Open with"
5. [ ] اختر "Sinan Note"
6. [ ] تحقق من استيراد الملف كملاحظة كود

### الخطوات - JavaScript:
1. [ ] أنشئ ملف test.js:
```javascript
function greet() {
    console.log("Hello World!");
}
```
2. [ ] افتح الملف من مدير الملفات
3. [ ] اختر Sinan Note
4. [ ] تحقق من الاستيراد

### الخطوات - JSON:
1. [ ] أنشئ ملف data.json:
```json
{
    "name": "Test",
    "version": "1.0"
}
```
2. [ ] افتح الملف
3. [ ] اختر Sinan Note
4. [ ] تحقق من الاستيراد

### النتيجة المتوقعة:
- ✅ التطبيق يظهر في قائمة "فتح بواسطة"
- ✅ الملف يُستورد كملاحظة جديدة
- ✅ نوع الملاحظة يتم اكتشافه تلقائياً (python/javascript/code)
- ✅ المحتوى كامل وصحيح

## 🐛 اختبارات الحالات الخاصة

### اختبار النسخ مع ملاحظة مقفلة:
1. [ ] افتح ملاحظة مقفلة
2. [ ] جرب "عمل نسخة"
3. [ ] تحقق: هل النسخة مقفلة أم لا؟

### اختبار النسخ مع ملاحظة بها تذكير:
1. [ ] افتح ملاحظة بها تذكير
2. [ ] اعمل نسخة
3. [ ] تحقق: النسخة يجب ألا تحتوي على تذكير

### اختبار فتح ملف كبير:
1. [ ] أنشئ ملف .py بحجم 1MB
2. [ ] حاول فتحه في التطبيق
3. [ ] تحقق من الأداء

### اختبار فتح ملف غير مدعوم:
1. [ ] حاول فتح ملف .exe أو .zip
2. [ ] تحقق: يجب ألا يظهر التطبيق في القائمة
